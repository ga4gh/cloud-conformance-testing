# GA4GH Cloud Conformance Testing

This repo will provide test suites to validate implementations of GA4GH Cloud APIs — the [**Tool Registry Service (TRS)**](https://github.com/ga4gh/tool-registry-service-schemas) , [**Workflow Execution Service (WES)**](https://github.com/ga4gh/workflow-execution-service-schemas), and [**Data Repository Service (DRS)**](https://github.com/ga4gh/data-repository-service-schemas) — against their respective OpenAPI specifications.

For more information about the GA4GH Cloud Work Stream developing these APIs, check out [**ga4gh.cloud**](https://github.com/ga4gh/wiki/wiki).
